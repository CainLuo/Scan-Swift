# Scan-Swift



## 介绍

Swift封装AVFoundation扫码以及识别二维码图片功能

* 可以生成与识别二维码、条形码
* 支持自定义识别区域
* 支持ScanView与ScanViewController的重载
* 支持获取相册图片识别(效果一般)



## 使用方式

### CocoaPods

```ruby
pod 'Scan-Swift'
```



#### Swift Package Manager

```ruby
.package(url: "https://github.com/CainLuo/Scan-Swift.git", .upToNextMajor(from: "1.0.0"))
```
